## 🚀 Overview

TicTac Arena is a competitive and interactive digital version of the classic Tic Tac Toe game. Built for speed, strategy, and fun, it transforms a simple childhood game into an engaging arena where players test their logic and reflexes.

Whether you're playing casually or aiming to dominate the leaderboard, TicTac Arena delivers a seamless and exciting gameplay experience.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/tictac-arena.git
cd tictac-arena
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

---

## 🎮 How to Play

1. Start a new match
2. Players take turns placing X or O
3. Align 3 symbols horizontally, vertically, or diagonally to win
4. Outsmart your opponent and dominate the arena
